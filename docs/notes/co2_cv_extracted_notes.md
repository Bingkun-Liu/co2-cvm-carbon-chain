The note says the project tried to use deep learning to predict the carbon
chain of objects and affect people's decisions. The workflow included:

1. Use ImageNet to train or use a computer-vision model for image
   classification and object-detection tasks.
2. Use Python to build an autoencoder model to extract useful features from
   ImageNet-style data and decode/reconstruct images to improve detection
   range and accuracy.
3. Use a seq2seq model to process the carbon-value production chain, learn
   relationships between different consumption processes, and predict another
   object's carbon chain.
4. Offer the data to an app team to support human-computer interaction and help
   users better estimate and manipulate carbon emissions.
